# 🎯 TLTQUIZ - Платформа управления квиз-мероприятиями

Минималистичная веб-платформа для автоматизации управления квиз-мероприятиями с чёрным дизайном и пастельными жёлтыми акцентами.

## 🎨 Дизайн

- **Цветовая палитра**: Чёрный фон (#1a1a1a), пастельный жёлтый (#E8D165), белый текст (#f5f5f5)
- **Минималистичный стиль**: Чистый, понятный интерфейс без перегруженности
- **Адаптивный дизайн**: Полная поддержка мобильных и десктопных версий

## 🌐 Страницы

### Общедоступные
- **Главная** (`/`) - Лендинг с обзором мероприятий, слайдер результатов
- **Расписание** (`/schedule`) - Список всех мероприятий с фильтрами
- **Вход** (`/login`) - Форма входа в аккаунт
- **Регистрация** (`/register`) - Создание нового аккаунта

### Для авторизованных пользователей
- **Профиль** (`/profile`) - Личный кабинет с регистрациями и командами
- **Чат** (`/chat`) - Общение в командах (WebSocket ready)

### Для администраторов
- **Админ-панель** (`/admin`) - Управление мероприятиями и аналитика
  - Управление событиями (создание, редактирование, удаление)
  - Аналитика и статистика
  - Просмотр регистраций
  - Экспорт отчётов (PDF/Excel)

## 🚀 Запуск

### 1. Установка зависимостей

```bash
pip install -r requirements.txt
```

### 2. Добавьте логотип

1. Скачайте логотип TLTQUIZ из группы VK
2. Поместите его в `static/images/logo.png`

### 3. Запуск приложения

```bash
python run.py
```

Приложение будет доступно по адресу: `http://localhost:5000`

## 📝 Примечания

### Цвета
- Основной: `#1a1a1a` (чёрный)
- Акцент: `#E8D165` (пастельный жёлтый)
- Текст: `#f5f5f5` (белый)
- Вторичный текст: `#b0b0b0` (серый)

### Характеристики дизайна
- Минимум 3 строк контекста при редактировании кода
- Совместимость с Flask Jinja2 шаблонизатором
- Чистый CSS без фреймворков (Bootstrap и т.п.)
- Поддержка всех современных браузеров

## 🔐 Безопасность

При переходе на production:
1. Измените `SECRET_KEY` в `config.py`
2. Установите `DEBUG = False`
3. Используйте переменные окружения для конфиденциальных данных
4. Настройте HTTPS
5. Добавьте CSRF защиту
